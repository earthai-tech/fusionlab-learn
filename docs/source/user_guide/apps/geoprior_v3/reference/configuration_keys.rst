=======================
Configuration key reference
=======================

This page maps GUI controls to store keys and to underlying pipeline parameters.

Recommended format:

- UI section name
- Store key (e.g., ``train.epochs``)
- Pipeline argument / config field
- Notes (units, safe range, default)

You can keep a small table per tab. If the table becomes large, split into one
file per tab under this folder.
