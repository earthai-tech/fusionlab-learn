GeoPrior v3 GUI
======================

GeoPrior v3 is the desktop GUI for running the GeoPrior subsidence/head
forecasting workflow end-to-end: dataset selection, experiment configuration,
preprocessing, training, tuning, inference, transferability evaluation, map
analytics, and results inspection.

This section documents the **GUI (v3.2)**. For the underlying model and
physics, see :doc:`/user_guide/models/pinn/geoprior/index`.
