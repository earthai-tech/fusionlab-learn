Results tab
===========

.. figure:: /_static/apps/geoprior_v3/09_results_tab.png
   :alt: Results tab.
   :width: 96%

   Results tab layout (example screenshot).

Purpose
-------

Browse outputs from preprocessing/training/tuning/inference/transferability and render plots and summaries inside the GUI.

Typical flow
------------

1. Choose or confirm inputs for this tab.
2. Adjust parameters (they are store-backed and persist).
3. Click **Run** and watch progress in the log panel.
4. Review outputs in the **Results** tab.
