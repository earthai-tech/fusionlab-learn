Tools tab
=========

.. figure:: /_static/apps/geoprior_v3/10_tools_tab.png
   :alt: Tools tab.
   :width: 96%

   Tools tab layout (example screenshot).

Purpose
-------

Utility tools that assist with scripting, batch generation, quick plots, and other developer/operator helpers.

Typical flow
------------

1. Choose or confirm inputs for this tab.
2. Adjust parameters (they are store-backed and persist).
3. Click **Run** and watch progress in the log panel.
4. Review outputs in the **Results** tab.

------------

Outputs
