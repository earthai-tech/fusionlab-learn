Transferability tab
===================

.. figure:: /_static/apps/geoprior_v3/07_transferability_tab.png
   :alt: Transferability tab.
   :width: 96%

   Transferability tab layout (example screenshot).

Purpose
-------

Evaluate cross-city transfer using baseline/xfer/warm strategies and generate a transfer matrix and detailed per-strategy reports.

Typical flow
------------

1. Choose or confirm inputs for this tab.
2. Adjust parameters (they are store-backed and persist).
3. Click **Run** and watch progress in the log panel.
4. Review outputs in the **Results** tab.

Key controls
------------

Document the important controls in this tab. Prefer describing them in terms of:

- **What it changes** (store key / config field)
- **Why you would change it**
- **Safe defaults**

Outputs
-------

Transferability CSV/JSON, per-run manifests, and a matrix panel for review.
